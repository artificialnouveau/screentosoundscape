### Option 1: Open PREVIEW versions (Recommended for Quick Testing)

- **PREVIEW-index.html** - Homepage
- **PREVIEW-team.html** - Team page
- **PREVIEW-support.html** - Support page

### Option 2: Open Main Pages
The main HTML files work the same way:

- **index.html** - Homepage
- **team.html** - Team page
- **support.html** - Support page
- **prototype.html** - Prototype page
- **sounds.html** - Sounds page
- **co-creation-sessions.html** - Co-creation sessions page
- **phase-1-report.html** - Phase 1 report
- **phase-2-report.html** - Phase 2 report
- **tutorial.html** - Tutorial page

### Option 3: Use a Local Web Server
For the best experience (if you want to deploy or test like a real website):

```bash
# Navigate to the project directory
cd /Users/ahnjili_harmony/Documents/GitHub/screentosoundscape/

# Start a simple Python HTTP server
python3 -m http.server 8000

# Open in browser: http://localhost:8000
```
